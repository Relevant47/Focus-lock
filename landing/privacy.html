Privacy Policy

Last updated: May 2026

Short version: FocusLock collects nothing. No account, no telemetry, no analytics, no payment data. Everything stays on your device. The only external request is to check for software updates.

1. What data we collect

FocusLock collects no personal data whatsoever. All application data — your focus profiles, session history, block lists, analytics, and settings — is stored locally on your device in:

  • Windows: %ProgramData%\FocusLock\
  • macOS: /Library/Application Support/FocusLock/

This data never leaves your device.

2. Software updates

FocusLock periodically checks for updates by contacting our update server and the GitHub API. This request includes:

  • Your operating system and CPU architecture (e.g. "windows-x86_64")
  • Your current FocusLock version number

No personally identifiable information is included. Your IP address is visible to the update server as part of the normal HTTP request, but it is not logged or stored.

3. No telemetry

FocusLock has no crash reporting, telemetry, analytics, or tracking of any kind. We receive no data about how you use the application, which sites you block, or how long your sessions are. The full source code is public on GitHub — you can verify this yourself.

4. No account required

FocusLock does not require an account, email address, or any personal information to download or use. There is no registration process.

5. No payments

FocusLock is completely free. We do not process payments and we do not collect any financial information.

6. Children's privacy

FocusLock does not knowingly collect any information from anyone, including children.

7. Changes to this policy

If we make material changes to this policy, we will update the "Last updated" date above. Changes will also be reflected in the GitHub repository.

8. Contact

Privacy questions: me@oscarpetrikas.com or open an issue on GitHub.
