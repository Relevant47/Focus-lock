Features

Blocking that actually works

Every feature is designed to be unbypassable — because half-measures don't build discipline.

🌐

Every browser, zero config

Blocks at the OS DNS level — Chrome, Firefox, Safari, Edge, Brave, Arc, Tor. No extension. No setup per browser.

🔄

Survives everything

Runs as SYSTEM/root. Closing the app does nothing. Rebooting does nothing. Blocks come back before your desktop loads.

💻

App blocking too

Block Steam, Discord, games — any app by process name. The kill loop runs every 2 seconds.

🔒

Hardcore Mode

Enable it at session start and nothing can end it early. Cryptographically committed, no override.

🤝

Friend Lock

Generate a token, send it to a friend, start your session. Only that token can end it early.

🍅

Pomodoro engine

Built-in Pomodoro timer. Blocks lift automatically during breaks in non-strict mode.

📅

Scheduled sessions

Set recurring schedules (cron syntax). Sessions start automatically even if the app is closed.

📊

Focus analytics

Streak tracking, weekly charts, focus score with streak multiplier, and a 52-week heatmap.

🎯

Motivational intercept

Your own message appears on the block page every time you try to visit a blocked site.

How it works

Serious blocking in three steps

1

Install the daemon

A privileged background service installs once and runs silently — starting at boot, before you open a browser.

2

Choose what to block

Pick from curated category packs or build custom block lists. Save as Focus Profiles for one-click reuse.

3

Start a session

The daemon takes over. Closing the app changes nothing. Blocks hold until the timer hits zero.

🔓

Fully open source

The full source code is on GitHub — every line of the daemon is auditable. Session state is HMAC-signed with a key that only the daemon can read. The daemon only does three things: edit the hosts file, kill blocked processes, and listen on a local IPC socket.
Before starting a session, generate a random token and send it to a friend. FocusLock stores only a SHA-256 hash — not even you can retrieve the original token. Only your friend (who has the plaintext) can unlock the session early.
